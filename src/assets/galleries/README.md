# Gallery Images

This folder contains galleries for the home page. Add your school images to each subfolder:

## Folder Structure

- **school_compound/** - School premises and buildings
- **students_learning/** - Classroom and academic activities
- **sports_recreation/** - Sports events and recreational activities
- **events_ceremonies/** - School events and ceremonies
- **student_life/** - Daily student activities and friendships

## How to Add Images

1. Add your images to each subfolder
2. Name them as: `image1.jpg`, `image2.jpg`, `image3.jpg`
3. Supported formats: JPG, PNG, WebP
4. Recommended size: 500x400px (aspect ratio 5:4)

Example:
```
school_compound/
  ├── image1.jpg
  ├── image2.jpg
  └── image3.jpg
```

The home page will automatically display these images in the carousels.
